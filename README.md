# Assembly Notes

Small lessons for comparing C++ source with the assembly generated by the compiler.

## Lesson 01: C++ to Assembly

- `lesson-01-cpp-to-assembly/main.cpp` contains a simple C++ program.
- `lesson-01-cpp-to-assembly/main.s` contains the generated assembly for that program.

Regenerate the assembly on this machine with:

```sh
clang++ -std=c++20 -O0 -S lesson-01-cpp-to-assembly/main.cpp -o lesson-01-cpp-to-assembly/main.s
```
